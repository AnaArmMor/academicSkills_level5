Books

  • "Game Programming Patterns" by Robert Nystrom: This book covers the foundational programming patterns used in game development, such as the game loop, the component pattern, and the state pattern.
  • "Unity in Action: Multiplatform Game Development in C#" by Joe Hocking: This book provides a practical guide to using the Unity game engine for developing games on multiple platforms, including desktop, mobile, and VR.
  • "Game Development Principles" by Alan Thorn: This book covers the fundamental principles of game development, including game design, programming, and asset creation.
  • "Introduction to Game Design, Prototyping, and Development: From Concept to Playable Game with Unity and C#" by Jeremy Gibson Bond: This book offers a comprehensive introduction to game development using Unity and C#, covering topics such as game mechanics, player input, and AI.
  • "Game Engine Architecture" by Jason Gregory: This book provides an in-depth guide to the architecture of modern game engines, covering topics such as rendering, physics, and networking.
  • "Level Up! The Guide to Great Video Game Design" by Scott Rogers: This book provides practical tips and advice for creating engaging and fun video games, covering topics such as game mechanics, storytelling, and user interface design.
